virtual

[Experiment mode] Enables the experiment mode

Parameters
pIDPrimitive ID
winSizeSliding window size
paramInput container with parameters
nTNumber of experiment times
store_genA flag indicating to store the generative process states. This should be used with caution, since memory allocation for large data may degrade performance. It is recommended to use e_getState to get the current state of the network, without storing information on the back-end side. Hence, the client front-end does the storage.
store_infA flag indicating to store the inference process states. Valuable only if the parameter store_gen is set true. Thus should be used only for debugging, since memory allocation for large data may degrade performance. All the computation steps for the back propagation trough time (BPPT) algorithm are stored.
